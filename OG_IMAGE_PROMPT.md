# TimeBudget - OG Image Generation Prompt

Use this prompt with any AI image generator (DALL-E, Midjourney, etc.) to create the OG image for social sharing.

## Specifications
- **Size:** 1200x630 pixels
- **Format:** PNG
- **Location:** `frontend/public/og-image.png`

## Prompt for AI Image Generator

```
A clean, modern, minimalist marketing image for a time management SaaS application called "TimeBudget". 

The image should be 1200x630 pixels with a white (#FFFFFF) background.

Main elements:
- Left side: The text "TimeBudget" in bold, modern sans-serif font (like Inter or Poppins) in dark charcoal (#1f2937)
- Below the title: A tagline "Gestiona tu tiempo como un presupuesto" in smaller text, dark gray (#4b5563)
- Right side: A simple, elegant illustration of a clock merging with a pie chart or budget visualization
- Accent color: Orange (#f97316) used sparingly for highlights, the clock hands, or chart segments
- Style: Flat design, no shadows, clean lines, professional SaaS aesthetic
- Mood: Productive, organized, modern, trustworthy

No text other than "TimeBudget" and the tagline. No photorealistic elements. Keep it simple and social-media ready.
```

## Alternative Simple Version (for tools with limited capabilities)

```
Minimalist white background. Left: "TimeBudget" in bold dark text. Right: Simple orange and gray clock icon with pie chart elements. Modern, clean, SaaS style. 1200x630px.
```

## Manual Creation Guidelines

If creating manually in Figma/Canva:

1. **Canvas:** 1200x630px, white background
2. **Title:** "TimeBudget" 
   - Font: Inter Bold or similar
   - Size: 72px
   - Color: #1f2937
   - Position: Left-center, ~100px from left edge
3. **Tagline:** "Gestiona tu tiempo como un presupuesto"
   - Font: Inter Regular
   - Size: 24px
   - Color: #6b7280
   - Position: Below title, 20px gap
4. **Icon:** Clock + chart hybrid
   - Size: ~300x300px
   - Position: Right-center
   - Colors: #f97316 (orange), #1f2937 (dark), #e5e7eb (light gray)
5. **Optional:** Subtle grid pattern or dots in background at 5% opacity
